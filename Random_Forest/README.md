# Random Forest Classification of Teaching Methods

## Project Overview
This project applies a Random Forest classification model to analyze student evaluation data collected using a Google Forms survey. The survey was conducted among students to evaluate how effectively different statistical methods were presented during the course.

The objective of the analysis is to identify which presentation characteristics most influence students’ perception of the best explained method.

## Dataset Description
The dataset contains responses collected from students evaluating presentations based on clarity, engagement, structure, and the usefulness of demonstrations.

## Variables Used
- explanation_clarity
- audience_engagement
- slides_effectiveness
- presentation_structure
- response_to_questions
- r_application_usefulness

Target variable:
- best_method_overall

## Methods
- Random Forest Classification
- Feature importance analysis
- Model evaluation
